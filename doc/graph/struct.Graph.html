pub fn get_costs( &self, starts: Vec<IntersectionID>, mode: Mode, public_transit: bool, start_time: NaiveTime, end_time: NaiveTime, ) -> HashMap<RoadID, Duration>

From a list of start intersections, floods out the graph for a mode until end_time is reached. Returns the time needed to reach each road within that range. This query is not precise about positions along a road.

pub fn new<F: FnOnce(&mut Vec<Road>), R: OsmReader>( input_bytes: &[u8], osm_reader: &mut R, modify_roads: F, timer: &mut Timer, ) -> Result<Graph>

Constructs a graph from OpenStreetMap data.

  • input_bytes: Bytes of an osm.pbf or osm.xml file
  • osm_reader: A callback for every OSM element read, to extract non-graph data
  • modify_roads: Runs before any routing structures are calculated. Use to modify access per mode.
