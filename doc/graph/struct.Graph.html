pub fn new<R: OsmReader>( input_bytes: &[u8], osm_reader: &mut R, post_process_graph: Box<dyn Fn(&mut Graph) -> Result<()>>, scrape_graph: Box<dyn Fn(&mut R, &Graph) -> Result<()>>, profiles: Vec<(String, Box<dyn Fn(&Tags, &LineString) -> (Direction, Duration)>)>, timer: &mut Timer, ) -> Result<Graph>

Constructs a graph from OpenStreetMap data.

  • input_bytes: Bytes of an osm.pbf or osm.xml file
  • osm_reader: A callback for every OSM element read, to extract non-graph data
  • post_process_graph: A callback to remove edges and intersections after initially importing.
  • scrape_graph: A callback to capture anything from osm2graph that’s otherwise lost. It can be stored on the osm_reader struct.
  • profiles: A list of named profiles. Each one assigns an access direction and cost, given OSM tags and a Euclidean center-line. If every profile assigns Direction::None, then the Road is completely excluded from the graph.
